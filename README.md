## 📂 Projetos no Repositório

### 🏋️ 1. Sistema de Academia (SistemaAcademia)
Um sistema completo para gestão de alunos e treinos que exercita a relação entre múltiplas entidades.
* **Conceitos Aplicados:** * **Composição:** Estudantes possuem planos e listas de treinos.
    * **Enums:** Uso de tipos enumerados para planos (BASIC, PREMIUM) e tipos de exercícios.
    * **Encapsulamento:** Atributos privados com métodos de acesso controlados.
    * **Collections:** Manipulação dinâmica de dados com `List` e `ArrayList`.

### 📑 2. Processamento de Contratos (Interface)
Exercício focado em abstração e desacoplamento de código.
* **Conceitos Aplicados:**
    * **Interfaces:** Uso de contratos para serviços de pagamento online.
    * **Inversão de Controle:** Código preparado para diferentes motores de cálculo de juros.
